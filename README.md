# Java Learning - Capgemini Training

This repository contains Java programming exercises and practice problems organized by topics.

## Structure

```
JavaLearning/
├── src/
│   └── CoreProgramming/
│       ├── Arrays/
│       │   ├── Level_1/    # Basic array operations
│       │   └── Level_2/    # Advanced array operations
│       ├── ControlFlows/
│       │   └── Level_1/    # Conditional statements and loops
│       ├── Methods/
│       │   ├── Level_1/    # Basic methods
│       │   ├── Level_2/    # Intermediate methods
│       │   └── Level_3/    # Advanced methods (NumberChecker utilities, OTP, Calendar, etc.)
│       ├── ProgrammingElements/
│       │   └── Level_1/    # Basic programming concepts
│       └── Strings/
│           └── Level_1/    # String manipulation
└── Practice_Week1/         # Week 1 practice problems
```

## Topics Covered

### Arrays
- Dynamic input handling
- Matrix operations
- Number analysis
- Student data management
- BMI calculations

### Control Flows
- Conditional statements
- Loops and iterations
- Number comparisons

### Methods
- Number utilities (prime, palindrome, armstrong, etc.)
- Unit conversions
- Mathematical calculations
- OTP generation
- Calendar display
- Euclidean distance

### Programming Elements
- Basic I/O operations
- Calculations and conversions
- Data type operations

### Strings
- String manipulation
- Pattern matching

## How to Run

Navigate to the specific file you want to run and compile it:

```bash
cd src/CoreProgramming/[Topic]/[Level]
javac FileName.java
java FileName
```
